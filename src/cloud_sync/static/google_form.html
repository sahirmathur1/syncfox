Add Google Drive account (service account)

Get the service-account JSON key
  1. Open IAM & Admin → Service Accounts in your pulseyeg-tembo project.
  2. Either pick an existing SA or click + CREATE SERVICE ACCOUNT (name cloud-sync-sa, no project roles needed).
  3. Click into the SA → KEYS tab → ADD KEY → Create new key → JSON → CREATE. A .json file downloads.
  4. Copy the SA's email — looks like cloud-sync-sa@pulseyeg-tembo.iam.gserviceaccount.com.
  5. Open Google Drive (or your Workspace shared drive). Right-click each folder you want to sync → Share → enter the SA email above → Editor → Send. The SA can only see folders you've shared with it.
  6. Upload the JSON file below. Done.

The JSON key never leaves Dodoma after upload — stored age-encrypted in the cloud-sync database; rclone reads it from /data/credentials/<remote>.json (mode 0600). Revoke any time at the same Cloud Console page.

Leave blank for personal Drive folders. Find it in the URL of the shared drive: drive.google.com/drive/folders/<id>.
