Add Dropbox account

One-time: create a Dropbox app

Dropbox uses OAuth — Syncfox needs an app's key + secret to start the handshake. You create the app once at dropbox.com/developers/apps, then paste the key/secret below. Future Dropbox accounts you add will reuse the same app.

  1. Click Create app → choose Scoped access → Full Dropbox (or App folder; both work) → name the app anything (e.g. syncfox).
  2. On the app settings page, under OAuth 2 → Redirect URIs, add: {{REDIRECT_URI}}
  3. Under Permissions, check at minimum: files.content.read, files.content.write, files.metadata.read, files.metadata.write, account_info.read. Submit.
  4. Back on Settings, copy App key and click Show next to App secret.
Syncfox stores the key/secret in app_settings (encrypted at rest if SYNCFOX_MASTER_PASSWORD is set). The secret is never displayed back to you.

Already configured? Re-run the OAuth handshake using the saved app.
