Add iCloud account

Why the real password and not an app-specific one? Apple's iCloud Drive API uses SRP authentication, which doesn't accept app-specific passwords. After this initial setup, rclone stores trust cookies + a trust-token so subsequent syncs don't need the password — expect to re-enter it once every ~30 days when the trust token rotates.
